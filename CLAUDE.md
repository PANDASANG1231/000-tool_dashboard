# 项目说明

这是一个简单的数据展示 Dashboard。

**主要功能:**
- 数据可视化图表
- 关键指标展示
- 简单的数据筛选和查询

**技术栈:**
- 前端框架: React
- 样式: Tailwind CSS
- 图表库: Recharts
- 数据: 模拟数据或简单 API

---

# 开发角色

## 前端开发
负责构建清晰、易用的界面:
- 创建可复用的组件
- 实现数据可视化
- 保持代码简洁易懂
- 确保响应式布局

## UI 设计
设计简洁的数据展示界面:
- 清晰的信息层级
- 易读的图表和数据
- 统一的色彩和间距
- 桌面优先,兼顾移动端

---

# 开发规范

## 代码风格
- 使用 Prettier 自动格式化
- 组件命名: `PascalCase` (如 `SalesChart`)
- 函数/变量: `camelCase` (如 `fetchData`)
- 文件名与组件名保持一致

## Git 提交
简单的提交格式:
```
类型: 简短描述

feat: 添加销售额趋势图
fix: 修复日期选择器显示问题
update: 更新仪表盘布局
```

## 组件结构
```
components/
├── Dashboard.jsx       # 主仪表盘
├── charts/
│   ├── SalesChart.jsx
│   └── UserChart.jsx
└── cards/
    ├── MetricCard.jsx
    └── SummaryCard.jsx
```
